# Laravel Typescript Generation

Use the `oi-laravel-ts` package to generate TypeScript interfaces from Laravel Eloquent models. The generated file is
located at `resources/js/types/interfaces.ts` by default and should not be modified manually.

- IMPORTANT: Activate `oilab-laravel-ts` when working with Inertia React client-side patterns or when you modify Laravel
  Eloquent models.
